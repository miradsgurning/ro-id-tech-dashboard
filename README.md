# 🇷🇴 🇮🇩 Romania-Indonesia Tech Data Insight Dashboard

## Project Overview
This project is a web-based data visualization application developed as part of a **Computer Science** scholarship portfolio for the Romanian Government Scholarship 2026.

## Tech Stack
- **Language:** Python
- **Framework:** Streamlit
- **Libraries:** Pandas (Data Wrangling), Plotly (Interactive Charts)

## Key Features
- **Data Transformation:** Processing raw World Bank CSV data into a visual format.
- **Interactive Visualization:** Dynamic line charts showing tech-export trends.
- **Performance Optimization:** Implemented caching for faster data loading.

## How to Run Locally
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
