# 🇷🇴 🇮🇩 Romania-Indonesia Tech Data Insight Dashboard
### Live Project: [Click Here to View the App](https://ro-id-tech-dashboard.streamlit.app/)

## Project Overview
This project is a web-based data visualization application developed as part of a **Computer Science** scholarship portfolio for the Romanian Government Scholarship 2026. It analyzes high-technology export trends between Romania and Indonesia, highlighting the digital synergy between the two nations.

## Tech Stack
- **Language:** Python
- **Framework:** Streamlit (Web Application Development)
- **Libraries:** Pandas (Data Wrangling), Plotly (Interactive Charts)
- **Version Control:** Git & GitHub

## Key Features
- **Data Transformation:** Processing raw World Bank CSV data into a visual format.
- **Interactive Visualization:** Dynamic line charts showing tech-export trends.
- **Performance Optimization:** Implemented caching for faster data loading.

## How to Run Locally
1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
